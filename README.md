## config tips

~/.tmux.conf

* caps lock = ctrl
* base pane index 1
* vim-tmux-navigation ( c-h c-j c-k c-l )
* prefix + prefix: last window
* splits like in vim ( prefix + w, prefix + s )
* tpm!
